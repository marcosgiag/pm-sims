El proyecto en 6 paquetes de trabajo

Todo lo que tenés que presupuestar, incluyendo lo que no genera código pero sí costo real:

📱
App Móvil

iOS + Android. Programar cortes, mapa de zonas del jardín, alertas en tiempo real, historial. El equipo usa GitHub Copilot.

⚙️
Backend & APIs

Lógica de negocio, base de datos de usuarios y cortes, API REST para la app y para la máquina física. Alcance bastante bien definido.

☁️
Infraestructura Cloud

Instancias AWS, RDS, S3, CDN, monitoreo. Precios públicos disponibles en la calculadora de AWS.

🔌
Integración IoT

Protocolo MQTT + WiFi/4G entre la app y la cortadora. Comandos en tiempo real. Primera vez que el equipo trabaja con hardware.

🔐
Capacitación Ciberseguridad

8 developers necesitan formación en secure coding antes de arrancar. Empresa externa dicta el curso. Requisito del área de seguridad corporativa.

🔍
Auditoría de Seguridad

Pentesting + revisión de código antes del lanzamiento. Empresa independiente. Requisito contractual del cliente.

Pregunta para arrancar: Si hoy cualquier dev puede escribir código con IA, ¿por qué seguimos teniendo incertidumbre en el estimado? ¿Qué paquetes te parecen más predecibles y cuáles más riesgosos?
